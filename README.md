* FUNCIONALIDADES - Version 2016.1.2.0
*
* SUMA Y RESTA: NUMA + NUMB = NUMC / NUMA>0 y NUMB>0; NUMA>NUMB
*		     	     -NUMC / NUMA<0 y NUMB<0; NUMA<NUMB
*
* Referencia:  
* 	     15+2=17
*	     15-2=13
*	     2-15=-13
*	    -15-2=-17
* ------------------------------------------------------------------------------
*
** MULTIPLICACION: NUMA*NUMB = NUNC / NUMA>0 y NUMB>0; NUMA<0 y NUMB<0
*			      -NUMC / NUMA<0 y NUMB>0; NUMA>0 y NUMB<0
*
* Ejemplo:
*	   15*2=30
*	  -15*-2=30
*	  -15*2=-30
*	   15*-2=-30
*
*------------------------------------------------------------------------------
*
** PORCENTAJE: NUMA%NUMB = NUNC 
*			      
* Ejemplo:
*	   20%200=40
*	   5%100=5
